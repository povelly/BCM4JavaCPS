This package and its subpackages present an example of BCM plug-ins used to implement a hash map component i.e., a component that offers the services of a hash map. The Java interface MapImplementationI declares the signatures of the services that pertain to a hash map as is used as an implementation interface. The component interface MapI, both offered and required, turns this implementation interface into a component one (in the sense of BCM). As uaual, ports and connector are implemented for this component interface.

The packages fr.sorbonne_u.cps.map.interfaces, fr.sorbonne_u.cps.map.components, fr.sorbonne_u.cps.map.deployments, fr.sorbonne_u.cps.map.connectors and fr.sorbonne_u.cps.map.ports proposes a standard implementation for the hash map component. The package fr.sorbonne_u.cps.map.withplugin and its subpackages modify the implementation of the hash map to use plug-ins.
